== Description ==

Its simply store the user information like name, email, comments and file like (doc, docx, pdf) into the database. User can access this form through the use of this short code [resume_upload_form].

On other hand user can also review the list of data with this shot code [resume_upload_form_list]

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the [resume_upload_form] shortcode to save data
4. Use the [resume_upload_form_list] shortcode for list of your data
